# SPEC: クロコ オフ会 準備チェック

## 目的

オフ会当日、参加者が会場到着後にインストール作業を円滑に開始できるよう、事前準備状況を確認するチェックリストアプリ。

## 対象ユーザー

リベシティ主催の Claude Code（クロコ）インストール伴走会（オフ会）参加者

## 設問仕様

### 表示タイミング

| フェーズ | 設問 | 対象 |
|--------|------|------|
| 最初から全て表示 | Q1〜Q7 | 全員 |
| OS選択（Q7）後に順番に表示 | Q8〜Q12 | OS依存 |

### 設問一覧

| # | 質問 | 選択肢 | 対象 |
|---|------|--------|------|
| 1 | 名札をつけていますか？ | はい／まだ | 全員 |
| 2 | 本日のリベチャットで問診に答えましたか？ | 答えた／まだ・知らない | 全員 |
| 3 | 作業前にパソコンを再起動しましたか？ | 再起動済み／していない／わからない | 全員 |
| 4 | クロコで使うGmailアカウントを決めていますか？ | 決めている／まだ迷っている | 全員 |
| 5 | パソコンの電源ケーブルを繋いでいますか？ | 繋いでいる／バッテリーのみ | 全員 |
| 6 | スマホを手元（横）に置いていますか？ | 置いている／カバンの中 | 全員 |
| 7 | パソコンの種類を教えてください | Mac／Windows | 全員 |
| 8 | macOSのソフトウェアアップデートは完了していますか？ | 完了している／まだ・わからない | Macのみ |
| 9 | デフォルトのWebブラウザはChromeに設定されていますか？ | はい／いいえ／わからない | 全員 |
| 10 | スクリーンショット（スクショ）の撮り方を知っていますか？ | 知っている／知らない | 全員 |
| 11 | ターミナル（Mac）/ PowerShell（Win）の開き方を知っていますか？ | 知っている／知らない | 全員（OS別テキスト） |
| 12 | Spotlight（スポットライト）は起動しますか？ | 起動する／起動しない／わからない | Macのみ |

### 分岐仕様

| 条件 | 動作 |
|------|------|
| OS = Mac | Q8（macOSアップデート）を表示してからQ9へ |
| OS = Windows | Q8をスキップしてQ9へ |
| OS = Mac かつ Q11回答後 | Q12（Spotlight）を表示 |
| OS = Windows かつ Q11回答後 | Q12をスキップして完了 |

### OS別テキスト切替

OS選択後、以下の設問テキスト・補足を自動更新する。

| 設問 | Mac | Windows |
|------|-----|---------|
| Q9 補足 | システム設定 → デスクトップとDock → デフォルトのWebブラウザ | 設定 → アプリ → 既定のアプリ → Webブラウザー |
| Q10 補足 | Command + Shift + 3（全体）/ 4（範囲） | Windows + Shift + S |
| Q11 質問文 | ターミナルの開き方を知っていますか？ | PowerShellの開き方を知っていますか？ |

## 回答時の補足表示

| 設問 | No / わからない 選択時 |
|------|----------------------|
| リベチャット問診 | 赤い警告（サポート不可の旨） |
| 再起動 | 青い案内（再起動を促す） |
| 電源接続 | 赤い警告（バッテリー切れリスク） |
| macOSアップデート | 赤い警告（確認方法を案内） |
| Chrome | 青い案内（スタッフへ相談） |
| スクショ | 青い案内（ショートカット表示） |
| ターミナル/PowerShell | 青い案内（開き方を表示） |
| Spotlight | 青い案内（別方法を案内） |

## 下書き機能

- 全設問の下（ページ末尾）に「📋 チャットに貼り付ける下書き」を常時表示
- 初期状態：プレースホルダーテキストを表示
- 回答のたびにリアルタイム更新
- 全問回答後：コピーボタンが有効化・🎉完了表示
- リセットボタンで最初からやり直し可能

## 非機能要件

- スマートフォン対応（レスポンシブ）
- 単一HTMLファイル構成（外部依存なし）
- Cloudflare Workers でホスティング
