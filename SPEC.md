# SPEC: クロコ オフ会 当日準備チェック

## 目的

オフ会当日、参加者が会場到着後にインストール作業を円滑に開始できるよう、事前準備状況を確認するチェックリストアプリ。

## 対象ユーザー

リベシティ主催の Claude Code（クロコ）インストール伴走会（オフ会）参加者

## 機能要件

### チェックフロー

- 質問を1つずつ順番に表示する（前の回答後に次が出現）
- 上部にプログレスバーを表示する

### 分岐仕様

| 条件 | 分岐 |
|------|------|
| OS = Mac | macOSソフトウェアアップデート確認を表示 |
| OS = Windows | macOSソフトウェアアップデート確認をスキップ |

### 回答時の補足表示

| 質問 | No選択時の表示 |
|------|----------------|
| リベチャット問診 | 赤い警告（サポート不可の旨） |
| ノートPC | 赤い警告（スタッフ相談） |
| 電源接続 | 赤い警告（バッテリー切れリスク） |
| スマホ | 青い案内（取り出しを促す） |
| 名札 | 赤い警告（受付スタッフへ） |
| macOSアップデート | 赤い警告（操作方法を案内） |
| Gmail | 青い案内（アカウント選択を促す） |
| スクショ | 青い案内（ショートカット表示） |
| ターミナル/PowerShell | 青い案内（開き方を表示） |
| Chrome | 青い案内（インストール案内） |

### サマリー機能

- 全項目回答後にチェック結果一覧を表示
- コピーボタンでリベチャットに貼り付け可能

## 非機能要件

- スマートフォン対応（レスポンシブ）
- 単一HTMLファイル構成（外部依存なし）
- Cloudflare Workers でホスティング

## デザイン

- 既存の問診チェックアプリ（claudecode-offkai-checkin）と統一したデザイン
- 背景色: #f5f0e8（クリーム）
- ヘッダー色: #c0392b（レッド）
- Yes ボタン: #27ae60（グリーン）
- No ボタン: #e07b4a（オレンジ）
