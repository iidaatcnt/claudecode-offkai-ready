🦀 クロコ オフ会 準備チェックしましょう

オフ会会場に到着したら、以下のチェックをしてください

こんにちは！オフ会へようこそ😊
スムーズにインストールを進めるため、以下の項目を順番に確認してください。
🏷️ 名札
名札をつけていますか？
※ オフィスのルールです。受付でお渡しした名札をつけてください。
受付スタッフに声をかけて名札を受け取ってください。
💬 リベチャット
本日のリベチャットで問診に答えましたか？
※ オフ会当日のリベシティチャットに問診が投稿されています。答えることでサポートがスムーズになります。
まず問診に答えてからチェックを進めてください。こちらの準備の都合上、事前に状況を把握できていないとサポートはできません。
🔄 再起動確認
作業前にパソコンを再起動しましたか？
※ 再起動しておくことで不要なプロセスが終了し、インストールがスムーズに進みます。
インストール作業を始める前に一度再起動しておきましょう。スタッフにお声がけください。
📧 Gmailアカウント
クロコで使うGmailアカウントを決めていますか？
※ Anthropicアカウントに紐付けるメインのGmailアカウントを1つ決めておきましょう。
複数のGmailがある場合は、クロコ専用にするアカウントを1つ選んでおきましょう。新規作成も可能です。
🔌 電源確認
パソコンの電源ケーブルを繋いでいますか？
※ インストール中にバッテリー切れになるとトラブルの原因になります。電源ケーブルを接続してから始めましょう。
バッテリーの残量が少ないとクロードコードのインストールができません。電源ケーブルをお持ちの方はすぐに接続してください。お持ちでない方はスタッフにご相談ください。
📱 スマホ準備
スマホを手元（横）に置いていますか？
※ Anthropicアカウント作成時にSMS認証が必要です。認証コードの有効時間が短いので、すぐ手が届く場所に置いておきましょう。
カバンからスマホを取り出して、PCの横に置いておきましょう。
💻 パソコン種類
パソコンの種類を教えてください
※ OSによって次の確認項目が変わります。
🍎 Mac専用
macOSのソフトウェアアップデートは完了していますか？
※ クロコ（Claude Codeデスクトップ版）はmacOS Tahoe 15以上で動作が保証されています。
「システム設定」→「一般」→「ソフトウェアアップデート」から確認できます。アップデート中は時間がかかるため、スタッフにご相談ください。
🌐 ブラウザ確認
デフォルトのWebブラウザはChromeに設定されていますか？
※ Mac: システム設定 → デスクトップとDock → デフォルトのWebブラウザ で確認できます。
Chromeをデフォルトに設定していない場合や、インストールされていない場合はスタッフにお声がけください。
🛠️ スキルチェック
スクリーンショット（スクショ）の撮り方を知っていますか？
※ エラー画面などをスタッフに共有するときに使います。
Mac: Command + Shift + 3（全体）/ Command + Shift + 4（範囲選択）
Windows: Windows + Shift + S（切り取り＆スケッチ）
🛠️ スキルチェック
ターミナル / PowerShellの開き方を知っていますか？
※ クロコのインストールにターミナル（Mac）またはPowerShell（Windows）を使います。
Mac: Spotlight（Command + Space）で「ターミナル」と検索
Windows: スタートメニューで「PowerShell」と検索
🍎 Mac専用
Spotlight（スポットライト）は起動しますか？
キーボードの Command + Space を押してみてください
※ ターミナルを開くときにSpotlightを使います。
Spotlightが起動しない場合はスタッフにお声がけください。別の方法でターミナルを開きます。
📋 チャットに貼り付ける下書き
🎉
準備チェック完了！
